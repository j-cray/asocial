# Asocial

## Description
A bespoke social media management tool designed to streamline content scheduling and analytics across multiple platforms, prioritizing user privacy and data ownership.

## Features
- **Multi-platform Support:** Post to Mastodon, Bluesky, and others.
- **Scheduling:** Automate posts for optimal times.
- **Analytics:** Privacy-respecting engagement tracking.
